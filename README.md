# wiz-frontend

# Running Application:
 -> Dependency Required:
    1) node
    2) npm
    3) angular cli
    
# Commands to run:
  ```
  -> git clone <url>
  -> cd wiz-frontend && cd wizfrontend/
  -> cd wiz-angular/ && npm install && ng build --prod && cd ..
  -> mvn clean install 
  -> mvn spring-boot:run
  ```
  # port started at http://localhost:8080
